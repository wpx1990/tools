# 常用的shell脚本

| 目录               | 功能                                 |
| ------------------ | ------------------------------------ |
| observe            | 观察某个服务器上的性能，并打印到日志 |
| operate-remote-ssh | 远程操作其他机器                     |
| pinglist           | 批量ping                             |
| start-time         | 打印服务器启动时间                   |
